# Data Cleaning and Analysis - Survey Project

This project demonstrates my ability to clean, organize, and summarize data from a survey dataset. It is fully human-created and showcases practical data handling skills.

**Tasks performed:**
- Removed duplicate entries
- Corrected typos and standardized text
- Filled or left missing values with clear reasoning
- Summarized key findings: average scores, gender counts, responses per location
- Created visualizations to illustrate patterns

**Tools used:** Excel / Google Sheets  
**Purpose:** Internship application for Data Wrangler Intern at Public and Private Development Centre (PPDC)
